# us-superstore-excel-analysis
Excel-based sales analysis of US Superstore dataset to identify sales trends, profit patterns, and regional performance insights.
roject Overview
This project analyzes the US Superstore dataset using Microsoft Excel to identify key business insights related to sales, profit, discount patterns, and regional performance.
Objective
To explore business performance and extract insights that can help improve:
Sales strategy
Profitability
Regional decision-making
Product category performance
Tools Used
Microsoft Excel
Pivot Tables
Charts & Dashboards
Data Cleaning & Formatting
Dataset Description
The dataset contains sales transactions including:
Order details
Customer information
Product categories
Sales, profit, and discount data
Region and state information
Key Analysis Performed:
📊 Sales vs Profit
📊 Sales & Profit by Category
📊 Sales & Profit by Segment
📊 Regional Comparison
Excel-based analysis of the US Superstore dataset provided clear business insights into sales performance, profitability, and regional trends. The project demonstrates strong skills in data cleaning, pivot tables, and dashboard creation using Excel.
